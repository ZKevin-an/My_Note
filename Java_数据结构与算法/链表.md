# 链表
```
有序列表，虽然使用的是一块连续的内存空间，但是每个节点不一定是连续储存的
最基本一定要有data域和next域
```

# 单向链表
```
主要考察如何进行增删改查，翻转（头插法）以及如何进行两个有序链表的合并（基本思路：创建一个新链表，一个一个的插入）
注意：基本上所有的操作都需要创建一个临时节点变量（辅助指针），用于递进搜索每一个节点
```

# 双向链表
```
在单向链表的基础上增加了一个pre，指向前一个节点，也是主要考察增删改查
```

# 环形链表（约瑟夫环，约瑟夫问题）
```
基本思路：创建一个节点，让first节点指向它，并且它的next指向自己，构成环形
遍历思路：需要一个辅助指针cur，遍历直到cur.next == first及遍历完成

约瑟夫问题：
输入参数：开始的位置，计算次数，节点总数（可以不需要）
思路：
1.创建一个辅助指针cur，指向first的前一个元素（方便出圈的操作）
2.让first和cur都到开始的位置（cur始终指向first的前一个位置）
3.每次到大计数次数使当前节点出圈，直到first==cur，即为剩下的最后一个节点
```
