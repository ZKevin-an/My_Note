# date
```
date：显示当前时间
date+%Y：显示当前年份
date+%m：显示当前月份
date+%d：显示当前是哪一天
date "+%Y-%m-%d %H:%M:%S" 显示年月日时分秒
date+%s：显示当前的时间戳（从1970年1月1号开始到目前经过的秒数）
date -d ‘1 days ago’|‘-1 days ago’：显示前一天|明天的时间
date -s 字符串时间：设置系统当前时间
ntpdate 时间服务器名称：同步时间
```

# cal(calendar)
```
cal：查看当前月的日历
cal 年份：查看该年份的日历
cal -y：查看当前年份的日历
```
