# 介绍
__vi__: 是Unix和类Unix系统中最通用的文本编辑器。  
__vim__: 是从vi发展而来的文本编辑器，性能更强大。

# 模式转换
```
          i,a,o               :,/              :q
编辑模式 <=======> 一般模式 <========> 命令模式 ---->  退出
           ESC                ESC
```
# 一般模式
```
i: 插入，进入编辑模式
u：撤回操作
yy：复制当前行
p：粘贴到光标的下一行
数字+p：粘贴到光标的下一行，重复数字遍
数字+y：复制当前行下的所有数字行
dd：删除当前行
数字+dd：
y+$：复制从当前光标开始到当前行最后位置的内容
y+^：复制从当前光标前一位到当前行最开头的内容
w：跳到下一个单词开头
y+w：复制当前光标所在单词
d+w：删除当前光标位置到单词的结束位置
d+$
d+^
x：剪切
shift+x：将光标之前的剪切，可以用作删除操作
r：单个字符替换
shift+r：连续替换
^：移动到行头
$：移动到行尾
e：跳到下一个词的词尾
b：上一个词的词头
gg（H）：到文本的开头
G（L）：到文本的末尾
数字+G：跳到数字行的开头
:+set nu：显示行号
:+set nonu：不显示行号
```
# 编辑模式
```
i：当前光标前
a：当前光标后
o：当前光标行的下一行
I：光标所在行的开头
A：光标所在行的结尾
O：光标行的上一行
```
# 命令模式
```
:w  保存
:q  退出
:wq 保存并退出
:q! 不保存强制退出
/要查找的词  n查找下一个，N查找上一个
:noh 取消高亮显示
:set nu 显示行号
:set nonu 关闭行号
:s/old/new 替换当前行匹配到的第一个old为new
:s/old/new/g 替换当前行匹配到的所有old为new
:%s/old/new 替换文档中每一行匹配到的第一个old为new
:%s/old/new/g 替换文档中匹配到的所有old为new
```
